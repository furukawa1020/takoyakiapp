# Platform Comparison

## Android版 vs Web版

### 🎮 Android版 (Unity + Rust)

**プラットフォーム**: Android 端末のみ

**技術スタック**:
- Unity (C#)
- Rust ネイティブ物理エンジン
- 3D グラフィックス
- シェーダープログラミング

**入力**:
- ジャイロセンサー (3軸)
- 加速度センサー
- カルマンフィルタ

**物理演算**:
- Rust ネイティブ実装
- PID制御
- ソフトボディシミュレーション
- 頂点変形

**ファイルサイズ**: 大きい (Unity ランタイム + ネイティブライブラリ)

**デプロイ**: APK ビルド、Google Play または手動インストール

---

### 🌐 Web版 (HTML5)

**プラットフォーム**: 全ブラウザ (PC, モバイル, タブレット)

**技術スタック**:
- HTML5 Canvas
- Vanilla JavaScript
- CSS3
- 依存ライブラリなし

**入力**:
- マウスドラッグ
- タッチ操作

**物理演算**:
- JavaScript 簡易実装
- 回転速度ベースのシェイピング
- 変形シミュレーション

**ファイルサイズ**: 約 20KB (HTML + JS + CSS)

**デプロイ**: GitHub Pages (自動)

---

## 比較表

| 項目 | Android版 | Web版 |
|------|-----------|-------|
| プラットフォーム | Android のみ | 全ブラウザ |
| インストール | APK 必要 | 不要 (URL アクセス) |
| ファイルサイズ | 大 (数十MB) | 小 (~20KB) |
| グラフィック品質 | 高 (3D) | 中 (2D Canvas) |
| 物理演算 | 高精度 (Rust) | 簡易版 (JS) |
| センサー | ジャイロ/加速度 | マウス/タッチ |
| 起動速度 | 遅い | 速い (即時) |
| オフライン | 可能 | 要キャッシュ |
| 更新 | 手動 | 自動 (ページリロード) |
| 開発規模 | 大 (64+ ファイル) | 小 (3 ファイル) |

---

## どちらを選ぶべきか？

### Android版がおすすめ:

- ✅ ジャイロセンサーを活用したい

- ✅ 今すぐ試したい
- ✅ PC でプレイしたい
